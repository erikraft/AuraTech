AuraTech é um projeto educacional da Etec para inspirar novos desenvolvedores.

Desenvolvido por um grupo de adolescentes com foco em aprendizado colaborativo, design profissional e tecnologia acessível.

Projeto Etec Desenvolvimento institucional

Missão AuraTech

Combinar educação e tecnologia por meio de uma vitrine eletrônica profissional e com espírito colaborativo.

Visão

Inspirar novos desenvolvedores

Mostrar como um site profissional pode ser construído com HTML, CSS e um pouco de JavaScript, mantendo o código organizado.

Valores

Compartilhar conhecimento

O projeto incentiva ensino colaborativo, desenvolvimento de habilidades e uso de código aberto para criação de soluções reais.

Objetivo

Criar um site realista

Essa loja de dispositivos funciona como base para novos projetos e serve de inspiração para lojas profissionais.

Participe do desenvolvimento

Substitua as imagens locais, adicione novos produtos e personalize o visual para seu próprio portfólio ou loja.
